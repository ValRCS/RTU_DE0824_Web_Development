Three ways of adding CSS to page

There are three ways to add CSS to a page:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

Inline CSS is added directly to the HTML element. It is not recommended to use inline CSS for styling a whole page.
